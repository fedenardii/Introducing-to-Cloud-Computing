## 📚 Corso trattato

### ☁️ Introducing to Cloud Computing

Focus sui concetti base del cloud computing, sui modelli di servizio e distribuzione, sulla virtualizzazione, sul networking cloud, sulla sicurezza e su alcuni servizi AWS fondamentali.

Argomenti principali:

- cos’è il cloud computing;
- casi d’uso principali del cloud;
- cloud provider;
- cloud pubblico, privato e ibrido;
- modelli IaaS, PaaS e SaaS;
- vantaggi e svantaggi del cloud;
- virtualizzazione e containerizzazione;
- scalabilità verticale e orizzontale;
- elasticità, autoscaling e load balancing;
- Virtual Private Cloud;
- subnet pubbliche e private;
- route table, Internet Gateway e NAT Gateway;
- security group e Network ACL;
- proxy, reverse proxy e load balancer;
- responsabilità condivisa nel cloud;
- crittografia, IAM, MFA e Zero Trust;
- logging, monitoraggio e risposta agli incidenti;
- panoramica di Amazon Web Services;
- EC2, Lambda, S3, VPC, CloudWatch, CloudTrail e IAM;
- sessioni pratiche su EC2, VPC custom, S3 e utenti IAM.

## ☁️ AWS

La parte finale della dispensa introduce Amazon Web Services come piattaforma cloud di riferimento.

Servizi trattati:

- **EC2** — macchine virtuali nel cloud;
- **S3** — object storage tramite bucket;
- **Lambda** — funzioni serverless event-driven;
- **VPC** — rete privata virtuale nel cloud;
- **IAM** — gestione di utenti, ruoli e permessi;
- **CloudWatch** — metriche, log e allarmi;
- **CloudTrail** — tracciamento delle chiamate API;
- **Elastic Load Balancing** — distribuzione del traffico;
- **Auto Scaling** — adattamento automatico delle risorse.
